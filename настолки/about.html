Об авторе проекта

Этот сайт разработан в рамках учебной практики по теме «Запуск проекта по оптимизации веб‑приложения. Магазин авторских настольных игр».

Основные акценты проекта:

  • понятная структура страниц (главная, каталог, информация, контакты);
  • лёгкая иерархия заголовков для SEO и доступности;
  • минимальное количество блоков, отвлекающих пользователя от каталога;
  • подготовка интерфейса к дальнейшей технической оптимизации (кэширование, минификация и т. д.).

Какие задачи оптимизации можно добавить

  1. Подключить минифицированные версии CSS и JS.
  2. Сгенерировать фавикон и добавить манифест для PWA.
  3. Реализовать загрузку изображений с атрибутом loading="lazy".
  4. Настроить сжатие gzip/brotli на стороне сервера.
  5. Добавить базовую аналитику для оценки скорости загрузки и поведения пользователей.
